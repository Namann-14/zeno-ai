## ğŸ›  Installation & Setup

### Prerequisites

Ensure you have the following installed:
- **Node.js** >= 20.x
- **pnpm** >= 10.x
- **Git**

### Step-by-Step Guide

**1. Clone the repository**
```bash
git clone https://github.com/Namann-14/zeno-ai.git
cd zeno-ai
```

**2. Install dependencies**
```bash
pnpm install
```

**3. Set up environment variables**
```bash
# Copy the example env files
cp apps/web/.env.example apps/web/.env.local
cp apps/widget/.env.example apps/widget/.env.local
cp packages/backend/.env.example packages/backend/.env.local
```

**4. Configure Convex backend**
```bash
# Initialize Convex (follow the prompts)
cd packages/backend
pnpm run setup
```

**5. Start the development servers**
```bash
# From the root directory
pnpm dev
```

**6. Open the applications**
- ğŸŒ **Web Dashboard**: [http://localhost:3000](http://localhost:3000)
- ğŸ”Œ **Widget**: [http://localhost:3001](http://localhost:3001)

## âš™ï¸ Environment Variables

Create `.env.local` files in the respective app directories:

### `apps/web/.env.local`
```env
# Clerk Authentication
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_test_xxxxx
CLERK_SECRET_KEY=sk_test_xxxxx

# Convex Backend
NEXT_PUBLIC_CONVEX_URL=https://your-project.convex.cloud

# Sentry Error Monitoring
SENTRY_AUTH_TOKEN=your_sentry_token
NEXT_PUBLIC_SENTRY_DSN=your_sentry_dsn
```

### `apps/widget/.env.local`
```env
# Convex Backend
NEXT_PUBLIC_CONVEX_URL=https://your-project.convex.cloud

# VAPI AI Voice
NEXT_PUBLIC_VAPI_PUBLIC_KEY=your_vapi_key
```

### `packages/backend/.env.local`
```env
# Google AI SDK
GOOGLE_GENERATIVE_AI_API_KEY=your_google_ai_key

# Clerk Backend
CLERK_SECRET_KEY=sk_test_xxxxx
```
